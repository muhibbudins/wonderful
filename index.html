Introduction

Wonderful adalah sebuah admin template gratis yang di buat dengan pondasi dari Bootstrap Framework dan jQuery beserta dependensinya. Wonderful memiliki bentuk struktur yang cukup unik dari foldering dan designnya, disini saya mengedepankan kebersihan pada konten dan lebih modern dibandingkan dengan admin template lain yang terlalu penuh dengan konten yang seharusnya tidak ada di tempatnya.

Sebelum melanjutkan saya meminta maaf bila saya akan terlambat untuk melakukan update dan maintenance karena beberapa urusan lain, tapi saya berjanji untuk terus maintain project ini meskipun dengan beberapa kesibukan yang ada. Dan yang pasti untuk di garis bawahi bahwa project ini akan selamanya open source. Silahkan bila anda ingin merubah, memodifikasi, mendistribusikan ulang dll. berdasarkan lisensi yang berlaku (MIT License).

Anda akan sangat mudah untuk mengkustom template ini karena saya sudah menyediakan source berupa SASS/SCSS beserta cara menggunakanya (baca lebih lanjut tentang Style Guide).

Folder Structure

Berikut adalah struktur folder yang saya buat, disini saya buat serapi mungkin dengan mengelompokan kegunaan dan bagian-bagianya, diantaranya :

root
|- css
|- fonts
|- img
|\
| |- favicon
|- js
|- partials
|- scss
|\
| |- components
| |- core
| |- helper
| |- icon
|- html file / another file type

Pada struktur diatas pada root terdapat beberapa folder diantaranya css, fonts, img, js, partials & scss dan untuk file template disini saya simpan di root semua (disini saya belum sempat menggunakan static compiler semacam jekyll atau lainnya mungkin nanti di versi selanjutnya ðŸ˜ƒ).

Berdasarkan struktur diatas berikut definisi dari masing-masing folder :

  • css : sebagai destinasi hasil konversi sass/scss file dan juga sebagai tempat file css dari dependensi yang template ini gunakan
  • fonts : sebagai tempat menyimpan fonts yang di gunakan oleh file css
  • img : sebagai tempat asset berupa image dan didalamnya ada folder favicon (anda bisa merubah logo pada title bar anda dengan merubah logo pada folder ini)
  • js : sebagai tempat semua script javascript yang berjalan di template ini
  • partials : sebagai tempat menyimpan part dari html file yang akan di load oleh javascript untuk di masukan ke dalam template yang berada pada fungsi di file ./js/load.js.
  • scss : sebagai tempat source file untuk style yang digunakan oleh template ini di dalamnya saya kategorikan lagi berdasarkan kegunaannya yaitu source component, core, helper & icon. Component berisi semua style seperti modal, table, form dll. Core berisi style yang global untuk semua halaman, Helper berisi kumpulan fungsi dan variable untuk digunakan oleh semua source style dan Icon berisi icon yg digunakan diantaranya Ion Icons & Font Awesome (baca lebih lanjut tentang Customize Icon)
